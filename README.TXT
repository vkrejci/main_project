Toto je výukový projekt, ve kterém studenti doplňují datové atributy a odpovídající
metody set a get.

Tento projekt zatím obsahuje jednu třídu a to Pocitac se dvěma datovými atributy: invCislo a mistnost

úkoly:
- doplňte metody getInvCislo() a getMistnost(),
- doplňte metodu
	public String toString()
  která vrátí řetězec s inv. číslem a místností,
- doplňte datový atribut „odpovědná osoba“ a metody setOdpOsoba a getOdpOsoba,
- doplňte datový atribut macAdresa a odpovídající metody set/get, MAC adresa je 12místné hex číslo,
- do metody setMacAdresa doplňte kontrolu, že zadaný parametr má 12 míst,
